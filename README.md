# 🚧 Página en Construcción - Landing Page

Una landing page moderna y atractiva para mostrar que tu sitio web está en construcción.

## ✨ Características

- **Diseño moderno y responsivo** - Se adapta a todos los dispositivos
- **Animaciones suaves** - Efectos visuales atractivos
- **Barra de progreso animada** - Simula el progreso de construcción
- **Partículas flotantes** - Efecto visual de fondo
- **Enlaces sociales** - Conecta con tus redes sociales
- **Gradientes dinámicos** - Cambio automático de colores de fondo
- **Efectos interactivos** - Hover y click effects

## 🚀 Cómo usar

### Desarrollo local con Vite (Recomendado)

1. **Instalar dependencias**:
   ```bash
   npm install
   ```

2. **Iniciar servidor de desarrollo**:
   ```bash
   npm run dev
   ```
   Esto abrirá automáticamente tu navegador en `http://localhost:3000`

3. **Ver cambios en tiempo real**: Cualquier cambio que hagas en los archivos se reflejará automáticamente en el navegador

### Uso tradicional

1. **Abrir la página**: Simplemente abre el archivo `index.html` en tu navegador
2. **Personalizar**: Edita los archivos según tus necesidades
3. **Subir a hosting**: Sube todos los archivos a tu servidor web

### Comandos disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Construye la versión de producción
- `npm run preview` - Previsualiza la versión de producción

## 🎨 Personalización

### Cambiar el mensaje principal
Edita el archivo `index.html` y modifica:
```html
<h1>Página en Construcción</h1>
<p class="subtitle">Estamos trabajando para traerte algo increíble</p>
```

### Cambiar colores
En `styles.css`, modifica las variables de color:
```css
/* Colores principales */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Agregar enlaces sociales
En `index.html`, actualiza los enlaces:
```html
<a href="https://twitter.com/tuusuario" class="social-link">
    <i class="fab fa-twitter"></i>
</a>
```

### Cambiar el email de contacto
```html
<a href="mailto:tuemail@dominio.com" class="social-link">
    <i class="fas fa-envelope"></i>
</a>
```
